Willkommen zum Barrierefreien Frontend Demo

Dieses Projekt demonstriert moderne Frontend-Entwicklung mit Fokus auf Barrierefreiheit, Sass und Gulp. Es zeigt, wie semantisches HTML, zugängliches CSS und JavaScript harmonisch zusammenarbeiten.

Über das Projekt

Diese Anwendung wurde entwickelt, um Best Practices für barrierefreie Webentwicklung zu zeigen:

  • Semantisches HTML5: Verwendung korrekter HTML-Elemente für bessere Struktur
  • WCAG 2.1 AA: Einhaltung der Richtlinien für Web-Accessibility
  • Tastaturnavigation: Vollständig mit der Tastatur bedienbar
  • Responsive Design: Funktioniert auf allen Geräten
  • Sass & BEM: Wartbarer und strukturierter CSS-Code

Barrierefreiheits-Features

Tastaturnavigation:

  • Tab / Shift + Tab: Zwischen interaktiven Elementen navigieren
  • Enter / Space: Buttons und Links aktivieren
  • Escape: Dialoge schließen (falls vorhanden)

Screenreader-Unterstützung:

  • Semantische HTML-Struktur mit korrekten Überschriftenhierarchien
  • ARIA-Labels und -Beschreibungen
  • Live-Regionen für dynamische Inhalte
  • Skip-Links für bessere Navigation
